Eclipse和IDEA快捷键对比

2019-09-15
  • IDE

之前的学习中一直使用的是Eclipse，可以说对Eclipse的快捷键已经很透了。最近准备适应IDEA的使用，也是应开发中可能出现更换IDE的需求，提前做好准备。一经上手，发现区别大了去了。不过IDEA真的是好用，不愧神器。

记录一些常用的快捷键，方便查阅：

搜索

功能 Eclipse快捷键 IDEA快捷键
搜索文本 Ctrl + F Ctrl + F
搜索方法、变量 Ctrl + O Ctrl + F12
搜索类 Ctrl + Shift + T Ctrl + N
搜索文件 Ctrl + Shift + T Ctrl + Shift + N
搜索所有引用处 Ctrl + Alt + H Alt + F7
搜索所有文本出现的位置 Ctrl + H Ctrl + Shift + F

编辑

功能 Eclipse快捷键 IDEA快捷键
自动代码补全 Alt + / Ctrl + J
自动代码生成(get、set等) ALT + SHIFT + S Alt + Insert
快速生成try catch SHIFT + ALT + Z CTRL + ALT + T
快速修复错误 Ctrl + 1 Alt + Enter
删除当前行 Ctrl + D Ctrl + X
复制到下一行 Ctrl + Shift + 上下 Ctrl + D
代码行上下移动 ALT + 上下键 CTRL + SHIFT + 上下键
注释/取消注释 Ctrl + / Ctrl + /
选中当前字 Ctrl + W
补全当前行 ALT+Shift+L Ctrl + Shift + Enter
调出最近复制的N份内容 Ctrl + Shift + V
格式化代码 Ctrl + Shift + F Ctrl + Alt + L
快速导包package Ctrl + Shift + O Ctrl + Alt + O
改名 Alt + Shift + R Shift + F6

跳转

功能 Eclipse快捷键 IDEA快捷键
显示类、方法说明 F2 Ctrl + Q
打开光标所在处的类或方法 F3 CTRL + B
查找变量来源 F4 F4
重新编译 CTRL+F9
